# NekoPlayer 交互设计文档

> 版本: v1.0.0  
> 日期: 2026-04-08  
> 设计工具: Compose Multiplatform

---

## 1. 设计原则

### 1.1 设计目标
- **简洁**: 减少认知负担，核心功能一键可达
- **沉浸**: 深色主题，聚焦音乐内容
- **一致**: 跨平台统一体验，iOS/Android 视觉统一

### 1.2 设计语言
| 维度 | 规范 |
|------|------|
| 主题 | 深色模式为主，支持浅色 |
| 圆角 | 组件 12dp，按钮 8dp，小元素 4dp |
| 间距 | 8dp 网格系统 |
| 字体 | 系统默认字体，动态类型支持 |

## 2. 色彩系统

### 2.1 主色调
```kotlin
// 主题色
val Primary = Color(0xFFFF69B4)        // B站粉红
val PrimaryVariant = Color(0xFFFF1493) // 深粉
val Secondary = Color(0xFF00D4FF)      // 咪咕蓝

// 背景色
val Background = Color(0xFF121212)     // 主背景
val Surface = Color(0xFF1E1E1E)        // 卡片/表面

// 文字色
val OnBackground = Color(0xFFFFFFFF)       // 主文字
```

## 3. 页面结构

### 3.1 应用架构
```
┌─────────────────────────────────┐
│           状态栏                 │
├─────────────────────────────────┤
│                                 │
│          页面内容               │
│        (Screen Content)         │
│                                 │
├─────────────────────────────────┤
│        MiniPlayer (64dp)        │ ← 条件显示
├─────────────────────────────────┤
│           导航栏                 │ ← 如需要
└─────────────────────────────────┘
```

### 3.2 页面清单

| 页面 | 路由 | 说明 |
|------|------|------|
| Splash | `/splash` | 启动页，2秒后跳转 |
| LoginEntry | `/login` | 登录入口，可选B站登录 |
| BiliLogin | `/bili_login` | B站扫码登录 |
| Search | `/search` | 首页，歌曲搜索 |
| PlaylistList | `/playlists` | 歌单列表 |
| PlaylistDetail | `/playlist/{id}` | 歌单详情 |
| NowPlaying | `/now_playing` | 全屏播放 |
| NowPlayingQueue | `/queue` | 播放队列 |

### 4.2 SearchScreen (首页)

**布局**
```
┌─────────────────────────┐
│ 🔍 搜索歌曲...    📋    │ ← 搜索栏 + 歌单入口
├─────────────────────────┤
│ 🩷 歌曲名称 - 歌手      │ ← 搜索结果列表
│    bilibili            │
├─────────────────────────┤

│    migu                │
├─────────────────────────┤
│         ...             │
└─────────────────────────┘
```

**搜索栏**
- 高度: 56dp
- 背景: `Surface` + 12dp 圆角
- 左侧: 搜索图标
- 右侧: 歌单按钮 (📋)
- 提示文字: "搜索歌曲..."

**搜索项**
- 高度: 72dp
- 内边距: 16dp horizontal, 12dp vertical
- 封面: 48dp 圆角 4dp
- 标题: `OnBackground`，16sp
- 副标题: `OnBackgroundSecondary`，14sp
- 来源标签: 12sp，对应颜色，圆角 4dp

**空状态**
- 图标: 搜索图标放大版
- 文字: "输入关键词开始搜索"
- 颜色: `OnBackgroundTertiary`

**长按菜单**
- 类型: BottomSheet
- 选项:
  - ➕ 添加到歌单
  - ⏭️ 下一首播放

### 4.5 NowPlayingScreen (全屏播放)

**布局**
```
┌─────────────────────────┐
│ ↓ 歌曲名...        🔄  │ ← 顶部栏
├─────────────────────────┤
│                         │
│                         │
│      [专辑封面]         │ ← 大封面 (280dp)
│      圆角 16dp          │
│                         │
│                         │
├─────────────────────────┤
│ 歌曲名称                │ ← 歌曲信息
│ 歌手名                  │
├─────────────────────────┤
│ ───────⚫────────       │ ← 进度条
│ 01:23      04:56        │
├─────────────────────────┤
│ ⏮️   ⏯️   ⏭️          │ ← 播放控制
├─────────────────────────┤
│ 🔁  📋  🔔              │ ← 底部功能
│ 模式  队列  定时        │
└─────────────────────────┘
```

**背景**
- 当前歌曲封面高斯模糊 + 暗色遮罩
- 动态渐变，随歌曲切换变化

**专辑封面**
- 尺寸: 280dp × 280dp
- 圆角: 16dp
- 阴影: elevation 8dp

**进度条**
- 高度: 4dp (可拖动区域 24dp)
- 已播放: `Primary` 色
- 未播放: `SurfaceVariant`
- 拖动球: 12dp 圆形

**播放按钮**
- 播放/暂停: 64dp 圆形，`Primary` 填充
- 上一首/下一首: 48dp 圆形，`Surface` 背景
- 图标: 白色

**底部功能区**
| 图标 | 功能 | 行为 |
|------|------|------|
| 🔄 | 播放模式 | 点击切换: 顺序→随机→单曲→顺序 |
| 📋 | 播放队列 | 跳转到队列页面 |
| 🔔 | 定时关闭 | 弹出时间选择器 |

**播放模式指示**
- 图标随模式变化:
  - 顺序: 🔁 (默认)
  - 随机: 🔀
  - 单曲循环: 🔂

- 尺寸: 48dp × 48dp
- 圆角: 4dp
- 左边距: 16dp

**歌曲信息**
- 标题: 14sp，`OnBackground`，单行

**控制按钮**
- 播放/暂停: 24dp 图标
- 下一首: 24dp 图标

**进度条**
- 高度: 2dp
- 位置: 组件底部
- 颜色: `Primary`

**交互**
- 点击播放条: 展开 NowPlayingScreen
- 按钮: 各自独立响应

## 5. 交互动效

### 5.1 转场动画

| 场景 | 动画 | 时长 |
|------|------|------|
| Splash → 首页 | 淡出 | 300ms |
| 页面切换 | 滑动 (左右) | 250ms |
| MiniPlayer 展开 | 共享元素 + 展开 | 300ms |
| BottomSheet 弹出 | 从底部滑入 | 200ms |
| Dialog 弹出 | 缩放 + 淡入 | 150ms |

### 5.2 微交互动画

| 元素 | 交互 | 效果 |
|------|------|------|
| 按钮 | 点击 | 缩放 0.95 → 1.0，100ms |
| 卡片 | 长按 | 轻微缩小 + 震动反馈 |
| 列表项 | 滑动 | 惯性滚动 |
| 进度条 | 拖动 | 实时更新 + 触觉反馈 |
| 封面 | 播放 | 缓慢旋转 (可选) |

## 6. 响应式设计

### 6.1 断点

| 设备类型 | 宽度 | 适配 |
|---------|------|------|
| 手机 | < 600dp | 单列布局 |
| 小平板 | 600-840dp | 歌单2-3列 |
| 大平板/TV | > 840dp | 歌单3-4列，侧边导航 |

### 6.2 TV 适配
- 所有可交互元素增加焦点状态
- 焦点边框: 2dp，`Primary` 色
- 支持遥控器方向键导航

## 8. 无障碍设计

### 8.1 规范
- 所有图标按钮添加 contentDescription
- 颜色对比度 ≥ 4.5:1
- 支持屏幕阅读器
- 支持动态字体大小

### 8.2 触摸目标
- 最小触摸目标: 48dp × 48dp
- 相邻目标间距: ≥ 8dp
