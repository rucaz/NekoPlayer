# NekoPlayer 使用说明书

## 🎵 应用简介

NekoPlayer 是一款跨平台音乐播放器，支持 Bilibili 和咪咕音乐搜索，提供本地歌单管理、播放队列、全局 MiniPlayer 等功能。

---

## 📱 界面导航

### 1. 搜索页面（首页）

**功能**：搜索歌曲、播放音乐、管理歌单

| 操作 | 说明 |
|------|------|
| 🔍 搜索框 | 输入歌曲名或歌手名，点击搜索 |
| 📋 歌单按钮（右上角） | 点击进入歌单列表页面 |
| 👆 点击歌曲 | 播放该歌曲 |
| ⏱️ 长按歌曲（400ms） | 弹出操作菜单：添加到歌单 / 下一首播放 |

**来源标记**：
- 🩷 **bilibili**（粉红色）- Bilibili 音乐
- 💙 **migu**（粉蓝色）- 咪咕音乐

---

### 2. 歌单列表页面

**功能**：管理本地歌单

| 操作 | 说明 |
|------|------|
| ➕ 新建歌单 | 输入歌单名称创建 |
| 👆 点击歌单 | 进入歌单详情 |
| 🗑️ 删除图标 | 删除整个歌单 |

---

### 3. 歌单详情页面

**功能**：查看歌单内歌曲、播放歌单

| 操作 | 说明 |
|------|------|
| ▶️ 播放全部 | 播放歌单内所有歌曲 |
| 👆 点击歌曲 | 播放该歌曲 |
| ⋮ 更多按钮 | 从歌单移除歌曲 |
| 🗑️ 删除图标（右上角） | 删除整个歌单 |

---

### 4. 播放页面（Now Playing）

**功能**：控制播放、查看进度、切换模式

| 控件 | 说明 |
|------|------|
| ⏮️ 上一首 | 播放上一首歌曲 |
| ⏯️ 播放/暂停 | 切换播放状态 |
| ⏭️ 下一首 | 播放下一首歌曲 |
| 🔄 播放模式（顶部） | 顺序播放 → 随机播放 → 单曲循环 |
| 📊 进度条 | 拖动跳转到指定位置 |

**播放模式**：
- 🔁 **顺序播放** - 按列表顺序播放
- 🔀 **随机播放** - 随机打乱播放顺序
- 🔂 **单曲循环** - 重复播放当前歌曲

---

### 5. 全局 MiniPlayer（底部播放条）

**触发条件**：开始播放歌曲后自动显示

| 操作 | 说明 |
|------|------|
| 👆 点击播放条 | 展开全屏播放页面 |
| ⏯️ 播放/暂停按钮 | 切换播放状态 |
| ⏭️ 下一首按钮 | 跳到下一首 |

## 📋 常用操作流程

### 创建歌单并添加歌曲

1. 在搜索页面点击右上角 **📋 歌单按钮**
2. 点击 **➕ 新建歌单**，输入名称
3. 返回搜索页面，**长按**想要添加的歌曲
4. 选择 **"添加到歌单"**
5. 选择目标歌单或新建歌单

### 播放歌单

1. 进入歌单列表页面
2. 点击进入目标歌单
3. 点击 **▶️ 播放全部** 按钮

### 设置下一首播放

1. 在搜索页面或歌单详情页 **长按** 目标歌曲
2. 选择 **"下一首播放"**
3. 歌曲将插入到队列的下一首位置

## ⚠️ 注意事项

### 播放链接获取

- 点击播放时应用会自动获取播放链接
- 首次播放可能需要 1-2 秒加载时间
- 如果获取失败，会显示错误提示

### 歌单数据

- 歌单数据保存在本地数据库
- 删除应用会丢失所有歌单数据
- 歌单封面会自动使用第一首歌曲的封面

### 播放队列

- 播放队列只在内存中，退出应用后清空
- 歌单数据会持久化保存

### 后台播放

- 支持后台播放（需开启通知权限）
- 锁屏状态下可通过通知栏控制播放

## 🐛 常见问题

**Q: 搜索不到歌曲？**
A: 检查网络连接，目前仅支持 Bilibili 和咪咕音乐源。

**Q: 添加到歌单失败？**
A: 检查是否已添加过该歌曲（一首歌不能重复添加到同一个歌单）。

**Q: 播放没有声音？**
A: 检查系统音量，或尝试暂停后重新播放。

**Q: MiniPlayer 不显示？**
A: MiniPlayer 需要队列中有歌曲才会显示，请先播放一首歌曲。

**Q: 上一首/下一首不工作？**
A: 确保队列中有多首歌曲，单首歌曲无法切换。
